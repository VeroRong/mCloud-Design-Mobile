Stepper

加减操作

Basic Example:

import { StyleSheet } from 'react-native'
import { Stepper } from 'mCloud-mobile';

export default () => {
    return (
        <View style={styles.warp}>
              <Stepper
                        max={200}
                        min={0}
                        unit="米"
                        step={2}
                        defaultValue={60}
                        onChange={(e) => {}}
                    />
        </View>
    )
}

const styles = StyleSheet.create({
    warp: {
        flex: 1,
        backgroundColor: '#E6E6E6',
    }
})

Stepper Props

属性说明类型默认值
min最小值number0
max最大值number100
step每次操作加减数number1
defaultValue默认值number50
value当前值number无
unit文字后面的单位String无
onChangechange事件触发的回调函数(e: Object): e.value无
